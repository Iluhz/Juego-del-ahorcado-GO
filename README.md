# Juego del Ahorcado en Go 

Juego clásico del ahorcado en la terminal, desarrollado en Go como parte de la materia de estructura de datos y aprender el lenguaje

## Requisitos

- Go 1.18 o superior
- Visual Studio Code (recomendado)

## Instalación
```bash
git clone https://github.com/Iluhz/Juego-del-ahorcado-GO.git
cd Juego-del-ahorcado-GO
```

## Ejecución
```bash
go run main.go
```

## Cómo jugar

- El juego selecciona una palabra aleatoria de una lista predefinida
- Debes adivinar la palabra letra por letra
- Tienes 6 intentos antes de perder
- Con cada error se dibuja una parte del ahorcado en la terminal
